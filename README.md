## ♔ О проекте

> *«Каждый ход — стратегия успеха»*

**Шахматный ход** — это интерактивный одностраничный сайт (Landing Page), где навигация между разделами реализована через метафору шахматной партии. Каждый скролл — это шахматный ход с анимацией взятия фигуры, открывающей новый информационный блок.
